## Endpoints

1. Método: 
   POST
2. URL: /chat
3. Descripción: 
   Se le manda un texto que proceso primero el backend, luego se manda a la API de chatgpt y se retorna la respuesta dada.

4. Parámetros: 
- Nombre del usuario
- Texto escrito por el usuario
